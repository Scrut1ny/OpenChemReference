#### General Procedure

**Preparation of Acid Mixture**
- Work under a fume hood with full protective gear.
- Mix concentrated sulfuric acid and concentrated nitric acid 1:1 in a separation funnel.
- Cool the mixture in an ice bath (below 25°C, preferably 10–15°C).

**Addition of Glycerin**
- Add glycerin dropwise to the cooled acid while stirring.
- Maintain temperature below 25°C.

**Reaction Monitoring**
- Stir and add glycerin slowly; prevent rapid temperature rise.
- If too hot, pause addition and let cool.

**Separation**
- Allow mixture to settle; nitroglycerin forms layer at the bottom.
- Separate carefully using a funnel.

**Washing**
- Wash with sodium bicarbonate (NaHCO₃) solution, then distilled water.

**Storage**
- Store in small, clearly labeled shock-absorbent containers in a cool, dark place.
